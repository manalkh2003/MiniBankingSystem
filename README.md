# Mini Banking System

A **console-based Java Banking System** demonstrating **OOP principles**: Inheritance, Polymorphism, and Encapsulation.

**Features:**  
- Saving, Current, and Loan accounts  
- Deposit, Withdraw, and Transfer operations  
- Input validation and duplicate account prevention  
- Polymorphic behavior for account-specific rules

**Run:** Compile and run `Main.java` in any Java IDE.
